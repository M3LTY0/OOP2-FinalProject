# OOP2-FinalProject

This project is a Library Management System.

It connects to a database to read, write, and delete data about Customers, Books, and Orders.

To use this system, follow the menu structure of the program to accomplish what you want to accomplish. The menu system is fairly straightforward and will hopefully be able to easily guide you through the tasks you're looking to do. Make sure to not enter incorrect values so that your experience of the program is smooth.
